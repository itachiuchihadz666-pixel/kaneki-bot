# Kaneki Bot 🤖

بوت فيسبوك ماسنجر متكامل مع لوحة تحكم ويب.

## المميزات

- ✅ أوامر إدارة المجموعة (طرد، قفل، منع المغادرة...)
- 🎨 توليد صور بـ Jimp (أمر /ابتيم)
- 🤖 ذكاء اصطناعي (Gemini / OpenAI)
- 🎵 تحميل أغاني يوتيوب
- 📚 مانغا وصور بينترست
- 🔒 نظام صلاحيات متعدد المستويات
- 💬 أوامر مخصصة قابلة للتخصيص
- ⚡ وضع المحرك (أداء أقصى)

## التثبيت

```bash
npm install
```

## التشغيل

```bash
node index.js
```

## متغيرات البيئة

| المتغير | الوصف |
|---------|-------|
| `GEMINI_API_KEY` | مفتاح Gemini AI (اختياري) |
| `OPENAI_API_KEY` | مفتاح OpenAI (اختياري) |

## ملف appstate.json

ضع ملف `appstate.json` بجانب `index.js` يحتوي كوكيز فيسبوك بتنسيق JSON.

## النشر على Railway

1. اربط الريبو بـ [Railway](https://railway.app)
2. أضف متغيرات البيئة
3. ارفع ملف `appstate.json` عبر لوحة التحكم
4. النشر تلقائي! 🚀

## الأوامر الرئيسية

| الأمر | الوصف |
|-------|-------|
| `/تشغيل` | تفعيل البوت |
| `/ايقاف` | تعطيل البوت |
| `/قفل` | قفل البوت للمشرفين |
| `/طرد @شخص` | طرد عضو |
| `/ابتيم` | عرض وقت التشغيل |
| `/تشغيل_المحرك` | وضع الأداء الأقصى |
| `/معلومات_المجموعة` | معلومات المجموعة |

## لوحة التحكم

يمكن تشغيل لوحة التحكم بشكل منفصل لإدارة البوت بواجهة رسومية.
